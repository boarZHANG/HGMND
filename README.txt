Hourly PM2.5 concentration records in Taiwan, 2019


This dataset contains hourly PM2.5 concentration records in Taiwan, 2019, from 67 air quality monitoring stations lying on the west half of the island.


The file is in the format of .rda, a data structure of language R. It contains a list of length 12. Each element in the list holds a data matrix containing monthly records, with the rows denote the hours and the columns denote the 67 stations.
